# SMS_Spam_Detection
Using Machine Learning algorithms.

## Technology Used
1. Onevsrestclassifier algorithm
2. Python
3. Flask
4. HTML/CSS

## Installation

1.Pandas

```
pip install pandas
```
2.Flask

```
pip install flask
```

## Running the python file

Unix Bash (Linux, Mac, etc.):
```
$ export FLASK_APP=apk
$ flask run
```
Windows CMD:
```
> set FLASK_APP=apk
> flask run
```
Windows PowerShell:
```
> $env:FLASK_APP = "apk"
> flask run
```
